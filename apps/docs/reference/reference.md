# MdrFrontEngine 文档

欢迎来到 MdrFrontEngine（以下简称 MFE）！

MFE 是一款**浏览器内运行**的面向现代 Web 开发的**可视化前端构建平台**。它以**页面**为基本组织单元，以**组件**为核心构建模块，支持从原型设计到生产部署的全流程开发。

MFE 旨在降低前端开发门槛，同时不牺牲灵活性与工程能力——无论是快速搭建静态站点，还是构建具备复杂交互逻辑的动态应用，均可在统一环境中高效完成。

## 核心特性

### 1. 组件化架构

- **内置组件库**：提供常用 UI 元素（按钮、表单、布局容器等）。
- **第三方集成**：无缝接入主流开源组件库（如 Ant Design、Material UI 等，具体支持列表见附录）。
- **自定义组件**：支持通过标准 Web Components 或框架特定语法（如 Vue SFC、React JSX）扩展组件体系。

### 2. 可视化逻辑编排（节点图系统）

- **触发器机制**：为组件实例绑定事件（如点击、加载完成），触发：
  - 预设操作（如跳转页面等）
  - 节点图
  - JavaScript 代码
- **节点图（Node Graph）**：通过图形化方式编写业务逻辑，支持：
  - 内置操作节点（跳转、状态更新、API 调用等）
  - 自定义 JavaScript 代码片段嵌入
  - 断点调试与单元测试
- **混合编程模型**：可以在节点图内插入原生代码，应对复杂逻辑或性能敏感场景。

### 3. 数据驱动开发

- 每个组件可关联**数据模型（Data Model）**，支持 JSON Schema 定义。
- 支持动态绑定：组件内容可响应外部数据源（如 REST API、本地 JSON 文件）。
- 内置数据流管理，简化“数据获取 → 渲染 → 交互更新”闭环。

### 4. 多目标代码导出

MFE 支持将项目导出为多种技术栈的完整可运行代码，包括：

| 导出类型       | 说明                                                  |
| -------------- | ----------------------------------------------------- |
| 原生 Web       | 纯 HTML/CSS/JS，支持按页面拆分或打包为 Web Components |
| Vue 3          | Composition API 风格，含 Vite 构建配置                |
| React / Preact | 函数式组件 + Hooks，支持 JSX                          |
| Angular        | 独立 Angular 应用，含模块与组件结构                   |
| SolidJS / Qwik | 响应式/渐进式框架适配                                 |
| htmx           | 声明式交互增强，适合渐进式增强场景                    |

所有导出代码均可在 MFE 内**在线预览、编辑、下载（ZIP）**，并支持一键推送至 GitHub / GitLab / Gitee 仓库（需授权）。

### 5. 一键部署

集成主流静态站点托管平台：

- **GitHub Pages**
- **Vercel**

用户可在 MFE 界面内完成构建与部署，无需离开开发环境。

### 6. 开源与社区生态

- **完全开源**：MIT 许可证，无付费墙、无隐藏功能。
- **社区共享**：支持发布/复用以下资源：
  - 完整网站模板
  - 独立组件
  - 可复用节点图逻辑模块
- 鼓励“代码即文档”的协作文化，促进最佳实践传播。
