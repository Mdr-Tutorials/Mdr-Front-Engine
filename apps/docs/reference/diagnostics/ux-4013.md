# UX-4013 多步骤流程缺少进度和当前位置

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4013 表示 多步骤流程缺少进度和当前位置。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

导入、发布、授权、AI apply、迁移或配置向导没有说明当前步骤、总步骤、完成状态或返回路径

## 建议操作

添加步骤指示、当前状态、剩余任务和返回/取消入口

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
