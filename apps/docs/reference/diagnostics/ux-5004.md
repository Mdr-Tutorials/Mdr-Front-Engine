# UX-5004 动效缺少 reduced motion 降级

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `visual`  |
| 可重试   | 否        |

## 含义

UX-5004 表示 动效缺少 reduced motion 降级。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

自动播放、循环、视差、缩放、闪烁或大幅位移动效没有 `prefers-reduced-motion` 降级

## 建议操作

提供 reduced motion 样式、关闭入口或静态替代
