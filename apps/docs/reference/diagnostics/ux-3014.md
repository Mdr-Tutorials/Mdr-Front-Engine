# UX-3014 打印或导出视图布局不可读

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `info`   |
| 阶段     | `layout` |
| 可重试   | 否       |

## 含义

UX-3014 表示 打印或导出视图布局不可读。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

打印、PDF、图片导出或分享预览中内容裁切、分页错误、背景丢失或关键状态不可读

## 建议操作

添加 print/export 专用布局、分页规则、背景策略和替代文本

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
