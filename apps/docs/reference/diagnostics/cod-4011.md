# COD-4011 sandbox 权限拒绝

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

COD-4011 表示 sandbox 权限拒绝。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户代码访问了 sandbox 未授权的全局对象、网络、存储、文件、DOM、计时器或其他受控能力

## 建议操作

移除受限 API，或切换到显式声明并允许该能力的运行目标

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
