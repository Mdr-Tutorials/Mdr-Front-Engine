# NGR-5001 断点目标不存在

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | NGR       |
| 范围     | 节点图    |
| 严重程度 | `warning` |
| 阶段     | `debug`   |
| 可重试   | 否        |

## 含义

NGR-5001 表示 断点目标不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

调试配置中的 breakpoint 指向不存在的 graph node

## 建议操作

删除失效断点或重新设置断点

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
