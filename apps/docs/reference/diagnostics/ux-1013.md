# UX-1013 语言或文本方向声明缺失

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `info`          |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1013 表示 语言或文本方向声明缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面、文档或文本区域缺少 `lang`、`dir` 或等价语言方向声明

## 建议操作

为页面或局部文本设置正确语言和方向

- 错误码和 requestId

- 触发该错误的操作
- 可复现时的最小文档或配置
