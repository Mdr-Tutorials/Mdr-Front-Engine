# COD-2010 重命名符号存在冲突

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `symbol`  |
| 可重试   | 否        |

## 含义

COD-2010 表示 重命名符号存在冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

rename、AI 修复或批量替换会让新符号名与当前作用域内已有符号、导入、宿主变量或保留名冲突

## 建议操作

换用不会冲突的名称，或先调整已有符号

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
