# API-2001 用户未登录

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | API      |
| 范围     | 后端/API |
| 严重程度 | `error`  |
| 阶段     | `auth`   |
| 可重试   | 是       |

## 含义

API-2001 表示 用户未登录。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

需要登录的接口没有有效会话

## 建议操作

重新登录后继续操作

- 触发该错误的操作
- 可复现时的最小文档或配置
