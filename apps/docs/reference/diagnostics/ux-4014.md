# UX-4014 重复输入或重复确认要求过多

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4014 表示 重复输入或重复确认要求过多。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

系统已知的信息要求用户重复填写、重复选择或重复确认，且没有安全、隐私或业务必要性

## 建议操作

复用已有数据、自动填充、记住选择，或解释为什么必须重复输入

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
