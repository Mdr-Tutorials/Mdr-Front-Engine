# UX-4004 空状态缺少下一步行动

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4004 表示 空状态缺少下一步行动。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

列表、表格、面板或资源区域为空时，只显示空白或无行动建议

## 建议操作

说明为什么为空，并提供创建、导入、清除筛选或返回入口

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
