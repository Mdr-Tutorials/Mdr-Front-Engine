# UX-5009 Skeleton 或占位内容与最终布局差异过大

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `info`   |
| 阶段     | `visual` |
| 可重试   | 是       |

## 含义

UX-5009 表示 Skeleton 或占位内容与最终布局差异过大。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

loading skeleton、placeholder 或 optimistic UI 与最终内容尺寸差异过大，导致布局跳动

## 建议操作

让占位结构接近最终布局，并为动态区域设置稳定尺寸

- 可复现时的最小文档或配置
