# AI-1001 Provider 配置缺失

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | AI         |
| 范围     | AI 助手    |
| 严重程度 | `warning`  |
| 阶段     | `provider` |
| 可重试   | 否         |

## 含义

AI-1001 表示 Provider 配置缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

AI 助手需要 provider，但未配置 baseURL、model 或 credential

## 建议操作

打开 AI 设置并完成 Provider 配置
