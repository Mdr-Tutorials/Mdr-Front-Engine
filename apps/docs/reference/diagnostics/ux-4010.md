# UX-4010 状态反馈与实际结果不一致

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `content` |
| 可重试   | 是        |

## 含义

UX-4010 表示 状态反馈与实际结果不一致。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

UI 文案显示成功、已保存、已同步或已发布，但公开状态或后端响应表明结果未完成

## 建议操作

修正状态映射，避免过早显示完成结果

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
