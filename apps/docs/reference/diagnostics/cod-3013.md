# COD-3013 生命周期 hook 与宿主阶段不匹配

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3013 表示 生命周期 hook 与宿主阶段不匹配。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段绑定到当前宿主不支持的生命周期阶段，例如在静态样式宿主中声明 mount/update/unmount hook

## 建议操作

改用当前宿主支持的生命周期，或把代码移动到支持该 hook 的组件、节点或运行目标
