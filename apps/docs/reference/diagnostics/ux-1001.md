# UX-1001 文本对比度不满足 WCAG

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1001 表示 文本对比度不满足 WCAG。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

可见文本与背景色的对比度低于目标 WCAG 等级要求

## 建议操作

调整文本色、背景色、字号或字重，直到对比度满足目标等级

- 错误码和 requestId

- 触发该错误的操作
- 可复现时的最小文档或配置
