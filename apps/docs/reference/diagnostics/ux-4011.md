# UX-4011 术语或行话缺少解释

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4011 表示 术语或行话缺少解释。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面使用专业术语、缩写、内部概念或错误码简称，但没有上下文解释、链接或帮助入口

## 建议操作

添加简短解释、tooltip、文档链接或更贴近用户任务的文案

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
