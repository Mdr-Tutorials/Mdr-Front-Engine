# UX-4005 必填、格式或约束说明缺失

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4005 表示 必填、格式或约束说明缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

表单字段有必填、格式、长度、范围或唯一性约束，但提交前没有可见说明

## 建议操作

在字段附近补充约束说明或示例

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
