# RTE-3002 Outlet 无法匹配子路由

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | RTE       |
| 范围     | 路由      |
| 严重程度 | `warning` |
| 阶段     | `outlet`  |
| 可重试   | 是        |

## 含义

RTE-3002 表示 Outlet 无法匹配子路由。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前 route chain 中没有可渲染的子路由内容

## 建议操作

检查当前路径和子路由配置
