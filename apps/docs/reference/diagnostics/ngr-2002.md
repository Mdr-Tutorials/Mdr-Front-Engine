# NGR-2002 端口类型不兼容

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | NGR     |
| 范围     | 节点图  |
| 严重程度 | `error` |
| 阶段     | `port`  |
| 可重试   | 否      |

## 含义

NGR-2002 表示 端口类型不兼容。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

输出端口类型无法赋给输入端口类型

## 建议操作

使用转换节点或连接兼容端口

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
