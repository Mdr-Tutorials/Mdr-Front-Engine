# UX-5001 非文本图形对比度不足

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `visual`  |
| 可重试   | 否        |

## 含义

UX-5001 表示 非文本图形对比度不足。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

图标、边框、焦点轮廓、图表元素或控件状态与相邻颜色对比不足，导致状态难以辨认

## 建议操作

调整颜色、粗细、背景或状态形状

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
