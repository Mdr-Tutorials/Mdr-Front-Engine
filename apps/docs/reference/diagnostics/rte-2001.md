# RTE-2001 路由目标组件不存在

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | RTE       |
| 范围     | 路由      |
| 严重程度 | `error`   |
| 阶段     | `resolve` |
| 可重试   | 否        |

## 含义

RTE-2001 表示 路由目标组件不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

路由指向的 component document 无法在 workspace 中找到

## 建议操作

重新选择路由组件或恢复缺失文档

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
