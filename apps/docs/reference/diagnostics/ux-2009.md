# UX-2009 手势交互缺少等价控件

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2009 表示 手势交互缺少等价控件。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

滑动、拖拽、长按、双指缩放或悬停才能完成的任务没有按钮、菜单或键盘等价路径

## 建议操作

添加显式控件或命令入口

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
