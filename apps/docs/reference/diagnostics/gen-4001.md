# GEN-4001 代码发射失败

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | GEN      |
| 范围     | 代码生成 |
| 严重程度 | `error`  |
| 阶段     | `emit`   |
| 可重试   | 是       |

## 含义

GEN-4001 表示 代码发射失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

目标代码生成、格式化或文件写入失败

## 建议操作

重试导出；若复现，携带错误码上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
