# ANI-1002 时间线 ID 重复

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | ANI        |
| 范围     | 动画       |
| 严重程度 | `error`    |
| 阶段     | `timeline` |
| 可重试   | 否         |

## 含义

ANI-1002 表示 时间线 ID 重复。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

同一 animation document 中存在重复 timeline id

## 建议操作

重命名重复时间线
