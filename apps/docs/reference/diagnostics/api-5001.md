# API-5001 数据库写入失败

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | API           |
| 范围     | 后端/API      |
| 严重程度 | `error`       |
| 阶段     | `persistence` |
| 可重试   | 是            |

## 含义

API-5001 表示 数据库写入失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

数据库事务、约束或连接导致写入失败

## 建议操作

稍后重试；若复现，联系维护者

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
