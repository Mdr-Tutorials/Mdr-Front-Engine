# WKS-1002 工作区快照损坏

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | WKS     |
| 范围     | 工作区  |
| 严重程度 | `error` |
| 阶段     | `load`  |
| 可重试   | 否      |

## 含义

WKS-1002 表示 工作区快照损坏。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

workspace snapshot 缺少必需结构或无法反序列化

## 建议操作

尝试从历史版本恢复，或联系维护者修复数据

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
