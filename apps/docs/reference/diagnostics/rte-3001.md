# RTE-3001 布局路由缺少 Outlet

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | RTE       |
| 范围     | 路由      |
| 严重程度 | `warning` |
| 阶段     | `outlet`  |
| 可重试   | 否        |

## 含义

RTE-3001 表示 布局路由缺少 Outlet。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

布局路由存在子路由，但对应组件没有可用 Outlet

## 建议操作

在布局组件中添加 Outlet，或调整路由层级
