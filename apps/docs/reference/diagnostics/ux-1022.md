# UX-1022 认证流程依赖认知测试且缺少替代

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1022 表示 认证流程依赖认知测试且缺少替代。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

登录、验证或关键操作要求记忆、解谜、识别图像、转写字符等认知测试，且没有替代方式

## 建议操作

提供密码管理器兼容、一次性链接、生物识别、支持粘贴或其他可访问替代

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
