# UX-2011 交互反馈只依赖 hover

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2011 表示 交互反馈只依赖 hover。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

关键说明、操作入口或状态反馈只有 hover 时可见，键盘和触摸用户无法获得

## 建议操作

提供 focus、click、可见文本或触摸可用的等价反馈

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
