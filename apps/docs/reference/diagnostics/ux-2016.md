# UX-2016 指针取消行为不安全

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2016 表示 指针取消行为不安全。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

按下鼠标或触摸时立即触发高影响操作，用户无法通过移动指针、松开外部或取消手势来避免执行

## 建议操作

将高影响操作放到 pointer up/click，或提供取消、确认、撤销机制
