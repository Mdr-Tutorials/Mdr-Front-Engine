# UX-3004 关键操作在目标断点不可见

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `error`  |
| 阶段     | `layout` |
| 可重试   | 否       |

## 含义

UX-3004 表示 关键操作在目标断点不可见。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

保存、提交、关闭、返回、继续或主要导航在某个目标断点不可见或不可触达

## 建议操作

调整移动端导航、sticky action、overflow 或布局优先级

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
