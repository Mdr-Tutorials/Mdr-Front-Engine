# UX-3009 组件响应式约束缺失

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3009 表示 组件响应式约束缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

固定宽高、绝对定位、不可换行文本或无 min/max 约束导致组件无法适配目标容器

## 建议操作

添加 min/max、aspect-ratio、container query、flex/grid 约束或换行策略

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
