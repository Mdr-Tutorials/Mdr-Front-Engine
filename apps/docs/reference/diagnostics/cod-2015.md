# COD-2015 泛型或类型参数无法满足约束

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `symbol`  |
| 可重试   | 否        |

## 含义

COD-2015 表示 泛型或类型参数无法满足约束。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

函数调用、组件泛型、NodeGraph 泛型端口或工具类型的类型参数不满足上界、默认值或推断约束

## 建议操作

调整类型参数、输入值或宿主字段类型
