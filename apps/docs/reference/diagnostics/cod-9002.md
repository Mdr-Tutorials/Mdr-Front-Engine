# COD-9002 代码诊断证据不足

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | COD           |
| 范围     | 用户代码      |
| 严重程度 | `warning`     |
| 阶段     | `environment` |
| 可重试   | 是            |

## 含义

COD-9002 表示 代码诊断证据不足。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

上游语言服务、worker、sandbox、导入器或 AI 修复链路报告了问题，但缺少稳定 code、source span、artifactId 或宿主上下文，无法归类为更具体的 COD 码

## 建议操作

刷新诊断或重新打开代码片段；若仍出现，携带当前代码片段位置和操作步骤上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
