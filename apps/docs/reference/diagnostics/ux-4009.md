# UX-4009 数字、日期或单位缺少上下文

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4009 表示 数字、日期或单位缺少上下文。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

价格、尺寸、时间、百分比、性能或计数没有单位、时区、精度或含义说明

## 建议操作

添加单位、时间范围、时区、比较基准或格式化说明

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
