# COD-3010 事件 handler 参数签名不匹配

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3010 表示 事件 handler 参数签名不匹配。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Blueprint 事件、DOM 事件、组件事件或 NodeGraph 事件入口绑定的 handler 参数数量、顺序、名称或类型与宿主事件契约不一致

## 建议操作

按当前事件定义调整 handler 参数，或重新选择匹配的事件入口
