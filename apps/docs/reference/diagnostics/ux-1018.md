# UX-1018 内容在缩放或重排后不可用

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1018 表示 内容在缩放或重排后不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

文本缩放、浏览器 zoom 或窄宽度 reflow 后，内容被裁切、重叠或关键操作不可达

## 建议操作

修复响应式布局、文本换行、容器约束和滚动策略

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
