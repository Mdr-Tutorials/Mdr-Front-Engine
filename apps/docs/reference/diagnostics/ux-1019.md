# UX-1019 文本间距调整后内容不可读

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `info`          |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1019 表示 文本间距调整后内容不可读。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

行高、字距、词距或段落间距被用户样式调整后，文本重叠、截断或控件不可用

## 建议操作

避免固定高度锁死文本，给内容区域留出可伸缩空间

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
