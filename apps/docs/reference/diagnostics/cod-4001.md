# COD-4001 用户代码运行时抛错

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 是        |

## 含义

COD-4001 表示 用户代码运行时抛错。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户代码片段在预览、调试、表达式求值、worker 或 sandbox 中执行时抛出异常或返回失败状态

## 建议操作

查看代码位置、输入上下文和运行时错误摘要后修复代码

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
