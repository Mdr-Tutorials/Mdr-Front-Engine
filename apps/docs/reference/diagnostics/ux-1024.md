# UX-1024 页面方向被锁定且无必要理由

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `info`          |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1024 表示 页面方向被锁定且无必要理由。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面或功能强制横屏/竖屏，且不是绘图、游戏、媒体或其他方向敏感任务

## 建议操作

支持双方向布局，或提供方向锁定原因和替代访问路径
