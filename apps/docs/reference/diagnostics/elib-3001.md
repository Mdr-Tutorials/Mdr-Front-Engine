# ELIB-3001 注册阶段没有可渲染组件

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | ELIB       |
| 范围     | 外部库     |
| 严重程度 | `error`    |
| 阶段     | `register` |
| 可重试   | 否         |

## 含义

ELIB-3001 表示 注册阶段没有可渲染组件。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

注册外部库时没有任何可用的可渲染组件

## 建议操作

检查扫描结果和注册策略，确认至少有一个组件可被注册
