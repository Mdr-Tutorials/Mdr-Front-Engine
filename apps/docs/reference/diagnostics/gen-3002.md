# GEN-3002 依赖许可证策略不满足

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | GEN          |
| 范围     | 代码生成     |
| 严重程度 | `warning`    |
| 阶段     | `dependency` |
| 可重试   | 否           |

## 含义

GEN-3002 表示 依赖许可证策略不满足。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

导出依赖不满足项目配置的 license policy

## 建议操作

替换依赖或调整许可证策略

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
