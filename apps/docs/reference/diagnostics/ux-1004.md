# UX-1004 交互控件缺少可访问名称

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1004 表示 交互控件缺少可访问名称。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

button、link、menuitem、tab 或自定义交互控件没有可访问名称

## 建议操作

添加明确按钮文本、图标按钮标签或关联说明

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
