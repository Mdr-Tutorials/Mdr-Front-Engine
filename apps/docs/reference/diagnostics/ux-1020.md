# UX-1020 输入目的或自动完成语义缺失

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `info`          |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1020 表示 输入目的或自动完成语义缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

常见个人信息、认证、地址、联系方式或支付字段缺少 autocomplete、inputMode 或等价输入目的声明

## 建议操作

为字段补充正确的输入目的、键盘类型或自动完成语义

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
