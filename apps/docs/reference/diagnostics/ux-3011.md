# UX-3011 320px 宽度下内容不可重排

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3011 表示 320px 宽度下内容不可重排。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

目标页面在 320 CSS px 宽度或项目最小宽度下无法单列重排，导致内容裁切或横向滚动

## 建议操作

调整布局断点、网格、容器宽度和文本换行策略
