# UX-2007 Loading 或异步状态不可感知

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 是            |

## 含义

UX-2007 表示 Loading 或异步状态不可感知。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

长耗时操作没有 loading、progress、skeleton、禁用状态或状态公告

## 建议操作

为操作入口和结果区域添加明确异步状态

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
