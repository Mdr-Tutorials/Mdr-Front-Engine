# COD-2004 共享符号环境过期

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `symbol`  |
| 可重试   | 是        |

## 含义

COD-2004 表示 共享符号环境过期。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前代码诊断基于过期的 Authoring Environment revision，可能不反映最新 Blueprint、NodeGraph、Animation 或 Workspace 状态

## 建议操作

等待编辑器重新索引，或手动刷新当前项目上下文

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
