# AI-5001 AI Command dry-run 失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | AI        |
| 范围     | AI 助手   |
| 严重程度 | `error`   |

## 含义

AI-5001 表示 AI Command dry-run 失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

AI 生成的 command 在应用前 dry-run 失败

## 建议操作

不应用该计划，重新生成或手动调整
