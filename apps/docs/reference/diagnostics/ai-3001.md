# AI-3001 Prompt 上下文为空

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | AI        |
| 范围     | AI 助手   |
| 严重程度 | `warning` |
| 阶段     | `prompt`  |
| 可重试   | 否        |

## 含义

AI-3001 表示 Prompt 上下文为空。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前路由、选中节点或 workspace 上下文不足以生成计划

## 建议操作

先选择要编辑的页面或节点

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
