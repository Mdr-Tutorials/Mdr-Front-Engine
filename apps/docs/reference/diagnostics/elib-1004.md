# ELIB-1004 未注册的外部库 ID

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | ELIB    |
| 范围     | 外部库  |
| 严重程度 | `error` |
| 阶段     | `load`  |
| 可重试   | 否      |

## 含义

ELIB-1004 表示 未注册的外部库 ID。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前操作引用了未在工作区中注册的外部库 ID

## 建议操作

重新选择已注册的外部库，或先完成外部库注册

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
