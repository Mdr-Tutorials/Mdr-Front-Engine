# GEN-2001 组件 Adapter 缺失

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | GEN       |
| 范围     | 代码生成  |
| 严重程度 | `warning` |
| 阶段     | `adapter` |
| 可重试   | 否        |

## 含义

GEN-2001 表示 组件 Adapter 缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

目标框架没有当前组件类型的 adapter

## 建议操作

替换为支持的组件，或添加组件导出适配

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
