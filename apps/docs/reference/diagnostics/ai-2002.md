# AI-2002 模型能力不满足当前任务

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | AI        |
| 范围     | AI 助手   |
| 严重程度 | `warning` |
| 阶段     | `models`  |
| 可重试   | 否        |

## 含义

AI-2002 表示 模型能力不满足当前任务。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前模型不支持结构化输出、足够上下文或所需模态能力

## 建议操作

切换到支持该能力的模型

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
