# UX-5005 闪烁或频闪风险

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `error`  |
| 阶段     | `visual` |
| 可重试   | 否       |

## 含义

UX-5005 表示 闪烁或频闪风险。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面或动画存在高频闪烁、强对比闪烁或大面积频闪风险

## 建议操作

降低闪烁频率、对比度或面积，或移除该动效

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
