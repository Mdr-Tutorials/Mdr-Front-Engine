# AI-2001 模型发现失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | AI        |
| 范围     | AI 助手   |

## 含义

AI-2001 表示 模型发现失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`{baseURL}/models` 请求失败或返回无法解析

## 建议操作

手动填写模型名，或检查 Provider 是否支持模型列表接口
