# UX-2012 操作结果缺少就地反馈

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `info`        |
| 阶段     | `interaction` |
| 可重试   | 是            |

## 含义

UX-2012 表示 操作结果缺少就地反馈。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

保存、复制、应用、导入、导出或生成操作完成后没有在触发区域或结果区域反馈

## 建议操作

在操作入口附近显示成功、失败或下一步状态

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
