# UX-5002 视觉层级无法支撑主要任务

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `info`   |
| 阶段     | `visual` |
| 可重试   | 否       |

## 含义

UX-5002 表示 视觉层级无法支撑主要任务。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面主要操作、当前状态或关键内容与次要元素层级过近，导致扫描困难

## 建议操作

调整尺寸、权重、间距、位置或色彩强调

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
