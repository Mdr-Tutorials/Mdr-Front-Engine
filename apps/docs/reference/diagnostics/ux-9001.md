# UX-9001 UX 检查器未知异常

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `error`   |
| 阶段     | `checker` |
| 可重试   | 是        |

## 含义

UX-9001 表示 UX 检查器未知异常。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

UX checker 执行中出现未分类异常，无法完成扫描

## 建议操作

重试扫描；若复现，携带错误码、目标页面和检查配置上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
