# EDT-5001 命令无法进入历史栈

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | EDT       |
| 范围     | 编辑器    |
| 严重程度 | `warning` |
| 阶段     | `command` |
| 可重试   | 否        |

## 含义

EDT-5001 表示 命令无法进入历史栈。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

编辑操作缺少 reverse ops，无法支持 Undo/Redo

## 建议操作

当前改动仍可保存，但可能无法撤销
