# RTE-4001 导航目标无法解析

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | RTE        |
| 范围     | 路由       |
| 严重程度 | `error`    |
| 阶段     | `navigate` |
| 可重试   | 否         |

## 含义

RTE-4001 表示 导航目标无法解析。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`navigate` action 的目标路径或 route id 无法解析

## 建议操作

检查事件绑定中的导航目标
