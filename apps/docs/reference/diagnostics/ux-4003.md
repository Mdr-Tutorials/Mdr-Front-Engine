# UX-4003 错误消息缺少修复建议

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4003 表示 错误消息缺少修复建议。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

错误消息只描述失败，没有说明用户可以如何修复、重试或上报

## 建议操作

补充下一步建议、相关入口、可重试说明或上报信息

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
