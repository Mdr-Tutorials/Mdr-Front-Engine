# ANI-3002 SVG Filter primitive 不存在

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | ANI     |
| 范围     | 动画    |
| 严重程度 | `error` |
| 阶段     | `track` |
| 可重试   | 否      |

## 含义

ANI-3002 表示 SVG Filter primitive 不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

svg-filter-attr track 指向不存在的 filter primitive

## 建议操作

检查 SVG Filter 定义或重新绑定 track
