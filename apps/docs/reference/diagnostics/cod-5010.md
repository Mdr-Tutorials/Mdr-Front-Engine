# COD-5010 语言服务 worker 初始化失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `compile` |
| 可重试   | 是        |

## 含义

COD-5010 表示 语言服务 worker 初始化失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

TypeScript、CSS、SCSS、GLSL、WGSL 或自定义语言服务 worker 无法初始化、加载编译器、加载规则集或建立通信

## 建议操作

刷新项目上下文后重试；若持续失败，携带语言模式、项目上下文和错误码上报

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
