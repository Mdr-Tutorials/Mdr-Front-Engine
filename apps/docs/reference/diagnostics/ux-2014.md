# UX-2014 定时消失内容缺少暂停或延长路径

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2014 表示 定时消失内容缺少暂停或延长路径。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

toast、提示、验证码、会话、自动轮播或限时操作在用户可处理前自动消失，且无法暂停、延长或重新打开

## 建议操作

增加暂停、延长、重新打开、历史记录或非限时替代
