# COD-5001 转译失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `compile` |
| 可重试   | 是        |

## 含义

COD-5001 表示 转译失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户代码片段在 TypeScript、JavaScript、CSS、SCSS 或表达式转译阶段失败

## 建议操作

修复代码语法、类型或不支持的语言特性后重试
