# RTE-1001 路由路径重复

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | RTE        |
| 范围     | 路由       |
| 严重程度 | `error`    |
| 阶段     | `manifest` |
| 可重试   | 否         |

## 含义

RTE-1001 表示 路由路径重复。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Route manifest 中存在重复 path

## 建议操作

修改重复路由路径

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
