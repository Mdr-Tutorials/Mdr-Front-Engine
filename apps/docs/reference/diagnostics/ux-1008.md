# UX-1008 ARIA role 与元素语义冲突

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1008 表示 ARIA role 与元素语义冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

元素的 role、ARIA 属性或可交互行为与原生语义冲突

## 建议操作

优先使用原生语义元素；必要时修正 role 和 ARIA 属性组合
