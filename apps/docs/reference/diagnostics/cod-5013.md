# COD-5013 目标运行模式不支持当前语言特性

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `compile` |
| 可重试   | 否        |

## 含义

COD-5013 表示 目标运行模式不支持当前语言特性。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前预览、sandbox、worker 或导出前作者态运行模式不支持代码片段使用的语法、模块格式、CSS 特性或 shader 能力

## 建议操作

改用当前运行模式支持的写法，或切换到支持该语言特性的目标模式
