# UX-4006 状态标签缺少可理解含义

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4006 表示 状态标签缺少可理解含义。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

badge、tag、chip、颜色点或图标状态没有文本含义或图例

## 建议操作

添加状态文本、说明、图例或 tooltip

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
