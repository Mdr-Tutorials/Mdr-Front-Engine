# UX-4001 可见控件文案不明确

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4001 表示 可见控件文案不明确。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

按钮、菜单项、链接或标题使用含糊文案，无法从上下文判断动作或目标

## 建议操作

使用具体动作、对象和结果描述文案

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
