# UX-2005 禁用控件缺少原因或替代路径

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `info`        |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2005 表示 禁用控件缺少原因或替代路径。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

关键操作被禁用，但用户无法理解禁用原因或下一步行动

## 建议操作

提供 tooltip、说明、校验信息或引导到可完成前置条件的位置

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
