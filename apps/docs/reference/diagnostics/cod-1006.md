# COD-1006 源码编码或文本范围非法

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | COD      |
| 范围     | 用户代码 |
| 严重程度 | `error`  |
| 阶段     | `parse`  |
| 可重试   | 否       |

## 含义

COD-1006 表示 源码编码或文本范围非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段包含无法解码字符、非法 surrogate、损坏换行，或上游诊断返回的 source range 超出片段边界

## 建议操作

重新输入或粘贴代码，移除异常字符后重试
