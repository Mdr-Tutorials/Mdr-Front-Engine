# COD-3003 代码访问了当前上下文不可用的能力

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3003 表示 代码访问了当前上下文不可用的能力。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段访问了当前 sandbox、运行目标、导出目标或宿主对象未声明支持的 capability

## 建议操作

移除不可用 API，或切换到支持该能力的运行目标

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
