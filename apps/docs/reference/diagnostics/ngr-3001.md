# NGR-3001 控制流连线形成非法循环

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | NGR     |
| 范围     | 节点图  |
| 严重程度 | `error` |
| 阶段     | `edge`  |
| 可重试   | 否      |

## 含义

NGR-3001 表示 控制流连线形成非法循环。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

控制流 edge 产生不受支持的循环

## 建议操作

移除循环连线，或使用显式循环节点
