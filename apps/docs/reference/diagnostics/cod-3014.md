# COD-3014 异步返回值不被宿主接受

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3014 表示 异步返回值不被宿主接受。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前宿主要求同步结果，但代码片段返回 Promise、async function、stream 或其他异步结果

## 建议操作

改为同步返回值，或把逻辑移动到支持异步执行的事件、worker 或资源模块中
