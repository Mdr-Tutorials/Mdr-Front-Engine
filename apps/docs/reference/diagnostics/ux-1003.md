# UX-1003 表单控件缺少可关联标签

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1003 表示 表单控件缺少可关联标签。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

输入框、选择器、开关、滑块或自定义表单控件没有可由辅助技术读取的稳定标签

## 建议操作

为控件添加可见 label、`aria-label` 或 `aria-labelledby`
