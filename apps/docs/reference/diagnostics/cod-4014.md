# COD-4014 用户代码返回不可序列化结果

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

COD-4014 表示 用户代码返回不可序列化结果。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

worker、sandbox、预览桥接或持久化边界收到 function、symbol、DOM node、循环引用、class instance 或其他不可序列化结果

## 建议操作

返回 plain object、array、string、number、boolean 或当前宿主明确支持的结构

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
