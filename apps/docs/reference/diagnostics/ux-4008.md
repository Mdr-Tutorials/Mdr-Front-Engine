# UX-4008 本地化文本缺失或混用异常

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4008 表示 本地化文本缺失或混用异常。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

同一界面混用占位 key、未翻译文本、错误语言或不可读编码

## 建议操作

补齐本地化资源，或标记允许的技术词汇和专名

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
