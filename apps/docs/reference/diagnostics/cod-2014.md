# COD-2014 外部库导出类型缺失或不可用

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `symbol`  |
| 可重试   | 是        |

## 含义

COD-2014 表示 外部库导出类型缺失或不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

已注册外部库可以运行，但缺少类型声明、导出元数据或 props/events 类型，导致作者态补全和类型校验降级

## 建议操作

为外部库补充类型声明、适配器 metadata 或手动声明 props/events 类型

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
