# UX-1023 焦点被固定层遮挡

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1023 表示 焦点被固定层遮挡。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前焦点元素被 sticky header、floating toolbar、toast、drawer 或软键盘部分或完全遮挡

## 建议操作

调整滚动定位、safe area、层级、焦点滚动或 overlay 布局
