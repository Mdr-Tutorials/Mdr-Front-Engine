# ANI-4001 Keyframe 时间不递增

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | ANI        |
| 范围     | 动画       |
| 严重程度 | `warning`  |
| 阶段     | `keyframe` |
| 可重试   | 否         |

## 含义

ANI-4001 表示 Keyframe 时间不递增。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

track keyframes 未按 `atMs` 升序排列

## 建议操作

调整关键帧顺序或让编辑器自动排序

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
