# NGR-4001 节点执行失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | NGR       |
| 范围     | 节点图    |
| 严重程度 | `error`   |
| 阶段     | `execute` |
| 可重试   | 是        |

## 含义

NGR-4001 表示 节点执行失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

单个节点运行时抛出异常或返回失败状态

## 建议操作

查看节点输入、输出和错误详情

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
