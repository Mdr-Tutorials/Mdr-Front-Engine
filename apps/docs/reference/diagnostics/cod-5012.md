# COD-5012 CSS/SCSS 预处理失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `compile` |
| 可重试   | 否        |

## 含义

COD-5012 表示 CSS/SCSS 预处理失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

CSS、SCSS 或 Mounted CSS 在变量解析、嵌套展开、mixin/include、import、PostCSS 或预处理阶段失败

## 建议操作

修正样式语法、变量、mixin、导入路径或不支持的预处理特性
