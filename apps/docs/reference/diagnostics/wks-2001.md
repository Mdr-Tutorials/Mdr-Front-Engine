# WKS-2001 能力协商不支持当前写入协议

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | WKS          |
| 范围     | 工作区       |
| 严重程度 | `error`      |
| 阶段     | `capability` |
| 可重试   | 否           |

## 含义

WKS-2001 表示 能力协商不支持当前写入协议。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

前端尝试使用后端未声明支持的文档级保存、intent 或 graph patch 能力

## 建议操作

刷新页面或升级服务端

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
