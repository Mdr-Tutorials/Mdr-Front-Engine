# WKS-3001 文档不存在

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | WKS        |
| 范围     | 工作区     |
| 严重程度 | `error`    |
| 阶段     | `document` |
| 可重试   | 否         |

## 含义

WKS-3001 表示 文档不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

保存、读取或 patch 的 `docId` 不存在

## 建议操作

刷新工作区文件树并重新选择文档

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
