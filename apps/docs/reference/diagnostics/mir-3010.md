# MIR-3010 列表渲染配置非法

## 快速信息

| 名称     | 说明        |
| -------- | ----------- |
| 前缀     | MIR         |
| 范围     | MIR 文档    |
| 严重程度 | `warning`   |
| 阶段     | `value-ref` |
| 可重试   | 否          |

## 含义

MIR-3010 表示 列表渲染配置非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`list.source`、`itemAs`、`indexAs`、`keyBy`、`arrayField` 或 `emptyNodeId` 不满足列表渲染契约

## 建议操作

检查列表数据源、别名、key 字段和空状态节点
