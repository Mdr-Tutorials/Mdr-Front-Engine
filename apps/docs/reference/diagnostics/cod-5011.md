# COD-5011 Source map 生成或映射失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `compile` |
| 可重试   | 是        |

## 含义

COD-5011 表示 Source map 生成或映射失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

转译后的诊断、运行时 stack、shader log 或预处理器错误无法稳定映射回原始代码片段 source span

## 建议操作

根据可见的代码片段和错误摘要手动定位；若反复出现，重新保存或拆分代码片段

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
