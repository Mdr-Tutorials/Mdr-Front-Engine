# UX-9002 UX 规则配置非法

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `error`   |
| 阶段     | `checker` |
| 可重试   | 否        |

## 含义

UX-9002 表示 UX 规则配置非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

项目配置的目标 WCAG 等级、断点、主题、规则集或豁免项格式非法

## 建议操作

修正 UX 检查配置后重新扫描

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
