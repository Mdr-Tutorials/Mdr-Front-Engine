# UX-9005 UX 规则被显式豁免

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `checker` |
| 可重试   | 否        |

## 含义

UX-9005 表示 UX 规则被显式豁免。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

某个 UX 规则命中，但项目、页面、节点或组件配置了带原因和范围的显式豁免

## 建议操作

检查豁免原因是否仍然有效，到期后重新评估
