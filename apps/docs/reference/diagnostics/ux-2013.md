# UX-2013 快捷键与保留快捷键冲突

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2013 表示 快捷键与保留快捷键冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面或组件快捷键覆盖浏览器、系统、屏幕阅读器、编辑器保留快捷键，或与同一作用域内其他命令冲突

## 建议操作

调整快捷键、限定作用域，或提供可配置快捷键

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
