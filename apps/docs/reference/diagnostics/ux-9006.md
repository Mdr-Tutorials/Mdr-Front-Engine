# UX-9006 UX 诊断需要人工复核

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `checker` |
| 可重试   | 否        |

## 含义

UX-9006 表示 UX 诊断需要人工复核。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

检查器发现高风险 UX 模式，但缺少足够上下文自动判定通过或失败，需要人工确认

## 建议操作

查看证据、确认设计意图，并选择修复、豁免或标记为已复核

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
