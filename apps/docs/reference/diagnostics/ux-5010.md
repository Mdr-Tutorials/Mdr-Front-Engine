# UX-5010 图表或数据可视化缺少可读编码

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `visual`  |
| 可重试   | 否        |

## 含义

UX-5010 表示 图表或数据可视化缺少可读编码。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

图表只靠颜色、无图例、坐标轴缺失、单位缺失或数据点无法理解

## 建议操作

添加图例、标签、单位、表格替代或交互说明

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
