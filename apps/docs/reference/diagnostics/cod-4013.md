# COD-4013 用户代码递归或循环超过限制

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

COD-4013 表示 用户代码递归或循环超过限制。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户代码递归深度、循环次数、微任务链或依赖求值链超过当前执行环境限制

## 建议操作

增加终止条件，拆分计算，或减少递归和循环复杂度

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
