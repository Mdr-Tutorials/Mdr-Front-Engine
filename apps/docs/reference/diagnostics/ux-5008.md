# UX-5008 高密度界面缺少分组或分隔

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `info`   |
| 阶段     | `visual` |
| 可重试   | 否       |

## 含义

UX-5008 表示 高密度界面缺少分组或分隔。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

大量字段、操作或数据在同一区域内缺少分组、分隔、标题或空间节奏

## 建议操作

使用 panel、group、divider、heading、tabs 或 progressive disclosure 组织内容
