# UX-3003 文本在容器内截断且无恢复路径

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3003 表示 文本在容器内截断且无恢复路径。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

标签、按钮、表头、错误信息或关键状态文本被截断，且没有 tooltip、展开或响应式换行路径

## 建议操作

调整布局、允许换行、缩短文案或提供完整文本查看方式

- 可复现时的最小文档或配置
