# API-1002 请求参数缺失

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | API       |
| 范围     | 后端/API  |
| 严重程度 | `error`   |
| 阶段     | `request` |
| 可重试   | 否        |

## 含义

API-1002 表示 请求参数缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

必需 path、query 或 body 参数缺失

## 建议操作

重新执行操作

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
