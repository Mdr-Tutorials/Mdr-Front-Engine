# API-4004 资源不存在或不可见

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | API          |
| 范围     | 后端/API     |
| 严重程度 | `error`      |
| 阶段     | `validation` |
| 可重试   | 否           |

## 含义

API-4004 表示 资源不存在或不可见。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

请求的用户、项目、绑定或开发入口不存在，或当前用户不可见

## 建议操作

返回列表刷新后重新选择资源

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
