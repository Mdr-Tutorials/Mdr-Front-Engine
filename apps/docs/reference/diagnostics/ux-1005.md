# UX-1005 标题层级跳跃或页面缺少结构标题

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `info`          |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1005 表示 标题层级跳跃或页面缺少结构标题。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面或区域标题层级无法形成可导航结构，或关键 section 缺少标题

## 建议操作

调整 heading 层级，确保页面和主要区域能被快速浏览
