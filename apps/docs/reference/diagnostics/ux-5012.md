# UX-5012 主题切换时出现短暂不可读闪烁

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `info`   |
| 阶段     | `visual` |
| 可重试   | 是       |

## 含义

UX-5012 表示 主题切换时出现短暂不可读闪烁。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

深浅色主题、品牌主题或高对比主题切换时，文本、图标或背景短暂处于不可读组合

## 建议操作

优化主题变量加载顺序、过渡策略和初始主题同步
