# UX-5003 主题变量组合导致状态不可读

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `visual`  |
| 可重试   | 否        |

## 含义

UX-5003 表示 主题变量组合导致状态不可读。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

某主题下文本、边框、图标、背景、selection 或 disabled 状态不可读

## 建议操作

调整主题 token、组件 token 或状态 token 映射

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
