# COD-2003 类型不兼容

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `symbol`  |
| 可重试   | 否        |

## 含义

COD-2003 表示 类型不兼容。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段的表达式、返回值、赋值或调用参数不满足当前符号或宿主字段的类型约束

## 建议操作

调整表达式类型、返回值或宿主字段配置

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
