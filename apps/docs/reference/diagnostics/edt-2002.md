# EDT-2002 拖拽会产生循环结构

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | EDT        |
| 范围     | 编辑器     |
| 严重程度 | `error`    |
| 阶段     | `dragdrop` |
| 可重试   | 否         |

## 含义

EDT-2002 表示 拖拽会产生循环结构。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户尝试把节点移动到自身或自身后代内

## 建议操作

选择其他目标位置

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
