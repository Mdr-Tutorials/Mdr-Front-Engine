# ANI-3001 Track 属性不支持

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `warning` |
| 阶段     | `track`   |
| 可重试   | 否        |

## 含义

ANI-3001 表示 Track 属性不支持。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前 runtime 或导出目标不支持该 track property

## 建议操作

改用支持的动画属性
