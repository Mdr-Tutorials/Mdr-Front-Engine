# ELIB-2001 扫描阶段未发现可渲染导出

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ELIB      |
| 范围     | 外部库    |
| 严重程度 | `warning` |
| 阶段     | `scan`    |
| 可重试   | 否        |

## 含义

ELIB-2001 表示 扫描阶段未发现可渲染导出。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

扫描外部库导出时没有发现可渲染组件

## 建议操作

检查外部库是否导出可渲染组件，或调整扫描路径和适配策略

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
