# API-3001 权限不足

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | API          |
| 范围     | 后端/API     |
| 严重程度 | `error`      |
| 阶段     | `permission` |
| 可重试   | 否           |

## 含义

API-3001 表示 权限不足。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前用户无权读取或修改目标资源

## 建议操作

请求资源所有者授权，或切换账号

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
