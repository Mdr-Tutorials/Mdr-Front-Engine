# COD-3002 代码片段返回值不满足宿主契约

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3002 表示 代码片段返回值不满足宿主契约。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

事件处理、表达式、computed value、shader entry 或动画计算片段的返回值不满足宿主协议

## 建议操作

按当前字段、事件、节点端口或动画 track 要求调整返回值

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
