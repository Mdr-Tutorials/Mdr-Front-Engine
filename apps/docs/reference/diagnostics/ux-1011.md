# UX-1011 焦点指示器不可见或对比不足

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1011 表示 焦点指示器不可见或对比不足。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

可聚焦元素没有可见 focus indicator，或指示器与相邻颜色对比不足

## 建议操作

添加清晰的 focus ring、outline 或其他稳定焦点样式

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
