# UX-1010 颜色是唯一的信息表达

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1010 表示 颜色是唯一的信息表达。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

错误、选中、成功、风险或分类信息只能通过颜色区分

## 建议操作

同时使用文本、图标、形状、边框或模式来表达状态

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
