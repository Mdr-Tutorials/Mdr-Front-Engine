# COD-3012 代码片段 owner 类型不支持当前宿主

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `binding` |
| 可重试   | 否        |

## 含义

COD-3012 表示 代码片段 owner 类型不支持当前宿主。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段声明的 owner kind、host kind 或 artifact kind 与实际挂载位置不匹配，例如把 shader 片段挂到普通事件字段

## 建议操作

重新选择支持该代码类型的挂载位置，或把片段转换为当前宿主支持的语言和模式
