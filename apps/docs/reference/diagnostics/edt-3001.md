# EDT-3001 Inspector 字段 schema 不可用

## 快速信息

| 名称     | 说明        |
| -------- | ----------- |
| 前缀     | EDT         |
| 范围     | 编辑器      |
| 严重程度 | `warning`   |
| 阶段     | `inspector` |
| 可重试   | 否          |

## 含义

EDT-3001 表示 Inspector 字段 schema 不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

选中节点没有可解析的 Inspector panel schema

## 建议操作

继续编辑其他基础字段，或更新组件库适配

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
