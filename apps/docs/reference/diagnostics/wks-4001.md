# WKS-4001 Workspace revision 冲突

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | WKS       |
| 范围     | 工作区    |
| 严重程度 | `warning` |
| 阶段     | `sync`    |
| 可重试   | 是        |

## 含义

WKS-4001 表示 Workspace revision 冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

客户端提交的 `workspaceRev` 落后于服务端

## 建议操作

拉取最新工作区后重新应用改动

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
