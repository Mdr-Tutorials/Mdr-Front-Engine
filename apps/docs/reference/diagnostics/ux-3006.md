# UX-3006 滚动容器嵌套导致操作困难

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3006 表示 滚动容器嵌套导致操作困难。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

嵌套滚动区域使用户难以到达内容、焦点滚动错位或滚轮事件被错误截获

## 建议操作

简化滚动层级，明确主滚动容器，并修复焦点滚动策略
