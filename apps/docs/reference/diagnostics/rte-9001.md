# RTE-9001 Route 未知异常

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | RTE       |
| 范围     | 路由      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 是        |

## 含义

RTE-9001 表示 Route 未知异常。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

路由匹配、Outlet 渲染或导航执行中出现未分类异常

## 建议操作

重试操作；若复现，携带错误码和路径上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
