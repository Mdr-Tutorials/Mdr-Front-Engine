# UX-3001 小屏视口出现不可访问横向溢出

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3001 表示 小屏视口出现不可访问横向溢出。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

移动或窄视口下正文、表单、工具栏或关键操作产生不可访问横向滚动

## 建议操作

调整响应式断点、换行、网格轨道或 overflow 策略
