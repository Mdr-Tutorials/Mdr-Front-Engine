# UX-1012 媒体缺少字幕、说明或控制

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1012 表示 媒体缺少字幕、说明或控制。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

音频、视频、自动播放媒体或时间型内容缺少字幕、文本说明、暂停或音量控制

## 建议操作

添加字幕、文字稿、媒体控制或禁用自动播放

- 当前项目或工作区 ID
