# API-4009 业务冲突

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | API          |
| 范围     | 后端/API     |
| 严重程度 | `error`      |
| 阶段     | `validation` |
| 可重试   | 否           |

## 含义

API-4009 表示 业务冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

请求本身合法，但与现有业务状态冲突，例如邮箱已注册

## 建议操作

修改输入或选择其他资源后重试

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
