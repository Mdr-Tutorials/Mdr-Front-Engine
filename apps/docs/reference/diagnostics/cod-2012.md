# COD-2012 符号解析结果不唯一

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `warning` |
| 阶段     | `symbol`  |
| 可重试   | 否        |

## 含义

COD-2012 表示 符号解析结果不唯一。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

同一个引用名在当前作用域内匹配多个候选符号，例如局部变量、route param、graph output、外部库导出或宿主注入变量同名

## 建议操作

使用更明确的名称、命名空间或导入别名

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
