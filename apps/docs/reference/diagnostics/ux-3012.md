# UX-3012 屏幕方向切换后布局或状态丢失

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 是        |

## 含义

UX-3012 表示 屏幕方向切换后布局或状态丢失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

设备横竖屏切换后，焦点、滚动位置、弹层、输入内容或关键操作状态丢失

## 建议操作

修复方向变化时的状态保持和布局恢复策略

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
