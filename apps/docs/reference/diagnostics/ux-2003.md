# UX-2003 指针或触摸目标尺寸过小

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2003 表示 指针或触摸目标尺寸过小。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

可点击、可拖动或可触摸目标小于项目目标尺寸或 WCAG 目标尺寸要求

## 建议操作

增大目标区域、增加间距，或提供等价大目标操作入口

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
