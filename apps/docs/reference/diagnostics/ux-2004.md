# UX-2004 交互状态缺失

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | UX            |
| 范围     | 用户体验      |
| 严重程度 | `warning`     |
| 阶段     | `interaction` |
| 可重试   | 否            |

## 含义

UX-2004 表示 交互状态缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

可交互元素缺少 hover、focus、active、selected、disabled、loading 或 pressed 等必要状态表达

## 建议操作

为控件补齐与功能相关的状态样式和语义属性

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
