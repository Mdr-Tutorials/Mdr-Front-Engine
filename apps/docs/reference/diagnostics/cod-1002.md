# COD-1002 不支持的语言模式

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | COD      |
| 范围     | 用户代码 |
| 严重程度 | `error`  |
| 阶段     | `parse`  |
| 可重试   | 否       |

## 含义

COD-1002 表示 不支持的语言模式。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段声明的 language 不在当前 Authoring Environment 支持列表中

## 建议操作

切换为当前功能支持的代码语言，或移除该代码片段

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
