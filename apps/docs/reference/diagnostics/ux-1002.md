# UX-1002 非文本内容缺少可访问替代

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1002 表示 非文本内容缺少可访问替代。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

有语义的信息图像、图标按钮、图表或媒体没有可访问名称、替代文本或等价说明

## 建议操作

添加 `alt`、`aria-label`、可见文本标签或图表摘要；纯装饰内容应明确标记为装饰
