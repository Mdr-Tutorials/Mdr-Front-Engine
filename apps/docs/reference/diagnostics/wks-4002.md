# WKS-4002 Route revision 冲突

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | WKS       |
| 范围     | 工作区    |
| 严重程度 | `warning` |
| 阶段     | `sync`    |
| 可重试   | 是        |

## 含义

WKS-4002 表示 Route revision 冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

客户端提交的 `routeRev` 落后于服务端

## 建议操作

刷新路由清单并重新保存
