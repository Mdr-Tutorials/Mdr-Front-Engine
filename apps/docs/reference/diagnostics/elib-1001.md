# ELIB-1001 加载失败（模块导入失败）

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | ELIB    |
| 范围     | 外部库  |
| 严重程度 | `error` |
| 阶段     | `load`  |
| 可重试   | 是      |

## 含义

ELIB-1001 表示 加载失败（模块导入失败）。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

外部库远程入口或模块导入失败

## 建议操作

检查外部库 URL、网络状态和模块导出配置，然后重试

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
