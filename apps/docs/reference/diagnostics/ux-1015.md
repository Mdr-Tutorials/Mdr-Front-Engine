# UX-1015 目标 WCAG 等级无法验证

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `info`          |
| 阶段     | `accessibility` |
| 可重试   | 是              |

## 含义

UX-1015 表示 目标 WCAG 等级无法验证。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

检查器缺少主题变量、字体尺寸、背景叠加或运行时状态，无法判断是否满足目标 WCAG 等级

## 建议操作

补齐主题变量、预览状态或检查配置后重新扫描
