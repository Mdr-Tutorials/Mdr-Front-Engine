# UX-4007 破坏性操作文案未说明影响范围

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4007 表示 破坏性操作文案未说明影响范围。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

删除、重置、覆盖、撤销发布等操作没有说明受影响对象或后果

## 建议操作

在按钮、确认框或详情中说明影响范围和恢复方式

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
