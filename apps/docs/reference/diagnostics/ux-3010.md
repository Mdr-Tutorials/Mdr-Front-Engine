# UX-3010 弹层位置在视口边缘不可达

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3010 表示 弹层位置在视口边缘不可达。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

menu、popover、tooltip、dropdown 或 context panel 超出视口且无法滚动访问

## 建议操作

添加 collision detection、flip、shift、max-height 或 portal 策略
