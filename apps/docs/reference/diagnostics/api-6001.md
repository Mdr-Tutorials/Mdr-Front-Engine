# API-6001 第三方集成调用失败

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | API           |
| 范围     | 后端/API      |
| 严重程度 | `error`       |
| 阶段     | `integration` |
| 可重试   | 是            |

## 含义

API-6001 表示 第三方集成调用失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

GitHub、OAuth 或其他外部平台 API 调用失败

## 建议操作

检查授权状态或稍后重试

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
