# UX-5011 图片文字缺少可访问替代

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `visual`  |
| 可重试   | 否        |

## 含义

UX-5011 表示 图片文字缺少可访问替代。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

关键信息以图片文字、截图文字或不可选择文本形式出现，且没有等价文本或真实文本替代

## 建议操作

使用真实文本，或提供等价说明、caption、alt text 或数据表

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
