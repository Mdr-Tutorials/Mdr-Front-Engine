# ELIB-1099 加载阶段未知异常

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | ELIB    |
| 范围     | 外部库  |
| 严重程度 | `error` |
| 阶段     | `load`  |
| 可重试   | 是      |

## 含义

ELIB-1099 表示 加载阶段未知异常。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

外部库加载阶段出现未分类异常

## 建议操作

重试加载；若复现，携带外部库配置摘要上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
