# API-4001 后端业务校验失败

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | API          |
| 范围     | 后端/API     |
| 严重程度 | `error`      |
| 阶段     | `validation` |
| 可重试   | 否           |

## 含义

API-4001 表示 后端业务校验失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

请求形状合法，但不满足业务规则

## 建议操作

根据提示修改输入后重试

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
