# UX-3005 阅读行宽或文本密度超出可读范围

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `info`   |
| 阶段     | `layout` |
| 可重试   | 否       |

## 含义

UX-3005 表示 阅读行宽或文本密度超出可读范围。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

长文本区域过宽、过窄、行距过低或密度过高，影响连续阅读

## 建议操作

设置最大宽度、行高、段落间距或内容布局

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
