# API-2002 会话已过期

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | API       |
| 范围     | 后端/API  |
| 严重程度 | `warning` |
| 阶段     | `auth`    |
| 可重试   | 是        |

## 含义

API-2002 表示 会话已过期。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

会话或 Token 过期

## 建议操作

重新登录

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
