# NGR-2001 必填输入端口未连接

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | NGR       |
| 范围     | 节点图    |
| 严重程度 | `warning` |
| 阶段     | `port`    |
| 可重试   | 否        |

## 含义

NGR-2001 表示 必填输入端口未连接。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

执行前发现 required input port 没有值或连线

## 建议操作

补齐端口输入或设置默认值

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
