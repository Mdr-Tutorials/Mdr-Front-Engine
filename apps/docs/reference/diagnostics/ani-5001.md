# ANI-5001 动画预览采样失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `error`   |
| 阶段     | `preview` |
| 可重试   | 是        |

## 含义

ANI-5001 表示 动画预览采样失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

预览器无法在指定时间采样动画状态

## 建议操作

检查绑定目标、track 值和 filter 定义

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
