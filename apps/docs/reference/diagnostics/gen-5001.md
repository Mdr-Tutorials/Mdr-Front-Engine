# GEN-5001 导出包生成失败

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | GEN      |
| 范围     | 代码生成 |
| 严重程度 | `error`  |
| 阶段     | `export` |
| 可重试   | 是       |

## 含义

GEN-5001 表示 导出包生成失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

项目脚手架、ZIP 或下载产物生成失败

## 建议操作

重试导出或选择较小范围导出

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
