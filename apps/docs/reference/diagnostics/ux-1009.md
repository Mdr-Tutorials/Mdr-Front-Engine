# UX-1009 状态变化未向辅助技术公告

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1009 表示 状态变化未向辅助技术公告。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

保存结果、表单错误、异步加载完成或动态列表变化没有 live region、focus move 或等价公告

## 建议操作

为关键状态变化添加 `role="status"`、`aria-live` 或明确焦点转移

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
