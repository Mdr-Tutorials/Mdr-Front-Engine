# COD-5002 Shader 编译失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | COD       |
| 范围     | 用户代码  |
| 严重程度 | `error`   |
| 阶段     | `compile` |
| 可重试   | 否        |

## 含义

COD-5002 表示 Shader 编译失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

GLSL 或 WGSL 代码片段无法通过 shader compiler 校验或编译

## 建议操作

根据 shader 编译日志修正入口函数、类型、uniform、binding 或语法
