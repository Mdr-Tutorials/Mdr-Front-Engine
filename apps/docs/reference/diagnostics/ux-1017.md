# UX-1017 缺少跳过重复内容的路径

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1017 表示 缺少跳过重复内容的路径。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面存在重复导航、工具栏或长列表，但键盘用户无法快速跳到主内容或关键区域

## 建议操作

添加 skip link、landmark、区域快捷导航或等价键盘命令

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
