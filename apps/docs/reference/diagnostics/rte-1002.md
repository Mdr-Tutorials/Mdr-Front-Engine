# RTE-1002 路由路径非法

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | RTE        |
| 范围     | 路由       |
| 严重程度 | `error`    |
| 阶段     | `manifest` |
| 可重试   | 否         |

## 含义

RTE-1002 表示 路由路径非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

路由 path 为空、缺少 `/` 前缀或包含不支持片段

## 建议操作

使用合法路径重新保存路由
