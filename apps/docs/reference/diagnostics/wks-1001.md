# WKS-1001 工作区不存在

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | WKS     |
| 范围     | 工作区  |
| 严重程度 | `error` |
| 阶段     | `load`  |
| 可重试   | 否      |

## 含义

WKS-1001 表示 工作区不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

请求的 workspace ID 无法解析到工作区快照

## 建议操作

返回项目列表并重新打开项目

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
