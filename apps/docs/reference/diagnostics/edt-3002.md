# EDT-3002 Inspector 字段写入被拒绝

## 快速信息

| 名称     | 说明        |
| -------- | ----------- |
| 前缀     | EDT         |
| 范围     | 编辑器      |
| 严重程度 | `error`     |
| 阶段     | `inspector` |
| 可重试   | 否          |

## 含义

EDT-3002 表示 Inspector 字段写入被拒绝。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Inspector 字段生成的 patch 被 schema、capability 或 graph validator 拒绝

## 建议操作

撤销该字段修改，检查输入值是否符合组件约束

- 可复现时的最小文档或配置
