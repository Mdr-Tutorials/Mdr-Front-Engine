# UX-5006 Disabled、selected 或 active 状态区分不足

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `visual`  |
| 可重试   | 否        |

## 含义

UX-5006 表示 Disabled、selected 或 active 状态区分不足。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

控件状态之间只存在极弱视觉差异，用户难以判断当前状态

## 建议操作

增强状态颜色、边框、图标、文本或形状差异

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
