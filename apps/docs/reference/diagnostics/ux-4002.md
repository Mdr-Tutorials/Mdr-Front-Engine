# UX-4002 链接文本无法说明目标

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `info`    |
| 阶段     | `content` |
| 可重试   | 否        |

## 含义

UX-4002 表示 链接文本无法说明目标。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

链接文本为“点击这里”“更多”“查看”等，脱离上下文无法判断目标

## 建议操作

改为说明目标页面、资源或动作的链接文本
