# UX-3002 内容被固定层或弹层遮挡

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3002 表示 内容被固定层或弹层遮挡。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

fixed header、toolbar、toast、modal、drawer 或 floating panel 遮挡关键内容或操作入口

## 建议操作

调整层级、间距、safe area、portal 位置或滚动定位

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
