# UX-1006 Landmark 或区域语义缺失

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `info`          |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1006 表示 Landmark 或区域语义缺失。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

页面缺少 main、nav、aside、header、footer 或等价 landmark，导致辅助技术无法快速跳转

## 建议操作

为页面主要区域添加语义元素或 landmark role

- 触发该错误的操作
- 可复现时的最小文档或配置
