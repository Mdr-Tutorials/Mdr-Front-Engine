# UX-3013 软键盘遮挡输入或主要操作

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3013 表示 软键盘遮挡输入或主要操作。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

移动端软键盘打开后，当前输入、提交按钮、错误信息或下一步操作被遮挡且无法滚动到可见区域

## 建议操作

添加键盘避让、scroll into view、sticky action 调整或 dynamic viewport 策略

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
