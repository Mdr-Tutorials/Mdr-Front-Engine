# API-9001 后端未知异常

## 快速信息

| 名称     | 说明          |
| -------- | ------------- |
| 前缀     | API           |
| 范围     | 后端/API      |
| 严重程度 | `error`       |
| 阶段     | `integration` |
| 可重试   | 是            |

## 含义

API-9001 表示 后端未知异常。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

后端处理请求时出现未分类异常

## 建议操作

重试操作；若复现，携带错误码和请求时间上报

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
