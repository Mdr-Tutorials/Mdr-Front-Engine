# UX-1007 ARIA 引用目标不存在

## 快速信息

| 名称     | 说明            |
| -------- | --------------- |
| 前缀     | UX              |
| 范围     | 用户体验        |
| 严重程度 | `warning`       |
| 阶段     | `accessibility` |
| 可重试   | 否              |

## 含义

UX-1007 表示 ARIA 引用目标不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`aria-labelledby`、`aria-describedby`、`aria-controls` 或 `aria-owns` 指向不存在的 ID

## 建议操作

修正引用 ID，或删除无效 ARIA 引用

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
