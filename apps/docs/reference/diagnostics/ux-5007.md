# UX-5007 可读字号或行高低于目标策略

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | UX       |
| 范围     | 用户体验 |
| 严重程度 | `info`   |
| 阶段     | `visual` |
| 可重试   | 否       |

## 含义

UX-5007 表示 可读字号或行高低于目标策略。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

正文、标签、表格、按钮或说明文字低于项目可读性策略设定的字号或行高

## 建议操作

调整 typography token、组件密度或断点样式

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
