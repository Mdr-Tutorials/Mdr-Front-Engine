# COD-2011 循环 import 或循环符号依赖

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | COD      |
| 范围     | 用户代码 |
| 严重程度 | `error`  |
| 阶段     | `symbol` |
| 可重试   | 否       |

## 含义

COD-2011 表示 循环 import 或循环符号依赖。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

代码片段、workspace module、external symbol 或 computed expression 之间形成循环依赖，导致解析或求值无法稳定完成

## 建议操作

拆分共享逻辑，移除循环 import，或把公共符号移动到独立模块

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
