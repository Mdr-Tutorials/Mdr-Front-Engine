# UX-3007 Safe area 或视口单位处理不完整

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | UX        |
| 范围     | 用户体验  |
| 严重程度 | `warning` |
| 阶段     | `layout`  |
| 可重试   | 否        |

## 含义

UX-3007 表示 Safe area 或视口单位处理不完整。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

移动端浏览器 chrome、刘海屏、软键盘或 dynamic viewport 导致内容被裁切

## 建议操作

使用 safe-area inset、dynamic viewport 单位或键盘避让策略

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
