# EDT-2001 拖拽目标非法

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | EDT        |
| 范围     | 编辑器     |
| 严重程度 | `warning`  |
| 阶段     | `dragdrop` |
| 可重试   | 是         |

## 含义

EDT-2001 表示 拖拽目标非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户尝试把节点放入不接受该节点类型或该位置的目标

## 建议操作

将组件拖到允许的容器或区域

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
