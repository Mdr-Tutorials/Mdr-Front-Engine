# GEN-3001 依赖包无法解析

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | GEN          |
| 范围     | 代码生成     |
| 严重程度 | `error`      |
| 阶段     | `dependency` |
| 可重试   | 是           |

## 含义

GEN-3001 表示 依赖包无法解析。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

codegen 无法为组件或外部库解析 package/import

## 建议操作

检查外部库配置和依赖版本

- 可复现时的最小文档或配置
