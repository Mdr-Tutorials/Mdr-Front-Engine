# COD-1001 代码解析失败

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | COD      |
| 范围     | 用户代码 |
| 严重程度 | `error`  |
| 阶段     | `parse`  |
| 可重试   | 否       |

## 含义

COD-1001 表示 代码解析失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户代码片段无法被当前语言 parser 解析，例如 TypeScript、JavaScript、CSS、GLSL、WGSL 或表达式语法错误

## 建议操作

根据 Code Editor 中的行列提示修正语法错误

- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
